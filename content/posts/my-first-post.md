## 欢迎

这是我的第一篇博客文章！我使用 Hugo 和 PaperMod 主题搭建了这个博客。

## 关于 Hugo

Hugo 是一个快速、灵活的静态网站生成器，具有以下优点：

- **速度快**：构建速度极快，即使是大型网站
- **易于使用**：简单的配置和内容管理
- **主题丰富**：有大量精美的主题可供选择
- **灵活强大**：支持自定义和扩展

## 开始写作

使用 Hugo 写作非常简单，只需要：

1. 创建 Markdown 文件
2. 添加 Front Matter（文章元信息）
3. 编写内容
4. 运行 `hugo server` 预览

就这么简单！

## 代码示例

Hugo 支持语法高亮，例如：

```python
def hello_world():
    print("Hello, Hugo!")

hello_world()
```

Hugo + GitHub Pages 是搭建个人博客的绝佳组合。期待在这里分享更多内容！
